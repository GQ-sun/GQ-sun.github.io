es6/node总结

es6

  • 箭头函数中this指向

    箭头函数内部是可以用this的，但这个this不是他自己的，而是这个函数在定义时所处的对象

  • let/const/var的区别

    let

    • 不能重复定义
    • 没有变量提升（var定义的变量是有变量提升的），必须先定义再使用
    • 全局变量不会附加到window对象的属性中
    • 具有块级作用域
    • 可以嵌套继承

    const

    ​ 是用来定义常量的，如果把一个对象声明一个常量，它的属性是可以改的，地址不能改

  • 解构赋值(数组,对象)–重要的

    数组规则：

    • 赋值符号左右两边都是数组，把右边的数组按下标从小到大取出来，放入左边的数组中。
    • 如果左边的变量并没有分配到某一个值：
      • 有默认值，使用其默认值。
      • 无默认值，其值是undefined

    对象规则：

    对于左边的变量名

    • 如果在右边的对象中能找出与左边属性名相同的项，则把右边的属性值赋值给左边的变量名。
    • 如果找不到，则看是否有使用默认值，有默认值就使用默认值，没有就是undefined。

    let {变量名1=默认值1，变量名2=默认值2} = {“属性名1”：属性值1,”属性名2”：属性值2,…}

  • array对象的拓展

    1.”…数组”,把数组中的元素一项项展开,把一个整体的数组拆开成单个元素

    ​ 应用1:数组拷贝

    ​ 应用2:数组合并

    ​ 应用3:Math.max()

    2.array.isArray();判断是不是数组

    3.array.from(东西);把一个东西转成数组(东西指的是伪数组或arguments)

    4.array.find(function(item,index,self){})从数组中找出我们符合条件的第一个元素

    5.array.findindex()从数组中找出我们符合条件的第一个下标

    元素找不到undefined,下标找不到-1

  • class类/实现继承

    语法：

    ​ class Father {}

    ​ class Son extends Father{}

  • set对象拓展

    成员的值是唯一的，没有重复的。

    成员方法

    size：属性，获取 set 中成员的个数，相当于数组中的 length
    add(value)：添加某个值，返回 Set 结构本身。
    delete(value)：删除某个值，返回一个布尔值，表示删除是否成功。
    has(value)：返回一个布尔值，表示该值是否为Set的成员。
    clear()：清除所有成员，没有返回值。
    forEach:遍历

  • 新增的数据类型以及含义

    1.Set 类似于数组，但数组可以允许元素重复，Set 不允许元素重复  

    2.Map 类似于对象，但普通对象的 key 必须是字符串或者数字，而 Map 的 key 可以是任何数据类型  

  • async/await的基本含义

    async 会将其后的函数（函数表达式或 Lambda）的返回值封装成一个
    Promise 对象

    await 会等待这个Promise 完成，并将其 resolve 的结果返回出来。

  • Promise的基本使用以及几个状态常见的含义—重要的

    执行一些异步的操作，在某个时间节点去调用resolve（），reject（）

    pending：待定（默认）

    resolved：已经解决的

    rejected：已拒绝的

node.js

  • npm是什么?如何去发布一个自己的包

    管理代码的包。

    1.注册npm账户

    2.选择目标文件夹，cd进去

    3.npm init

    4.配置完会生成package.json文件，然后写包的代码

    5.发布包 npm publish 包名

  • 除了npm还是用过其它的包管理工具吗?

    Yarn,bower,webpack,gulp,grunt

  • node中的模块是什么,什么是模块加载机制?

    每个模块都是一个独立的文件。每个模块都可以完成特定的功能，我们需要时就去引入它们，并调用。不需要时也不需要管它。

    分为 ：核心模块 自定义模块 第三方模块

    模块加载机制：先看一下缓存中是否已经引入过这个模块，如果有则用缓存中的，如果没有则新引入这个模块并放入缓存中。如果引入的模块只写了模块名，那么先看一下是不是核心模块，不是核心模块就到node_modules文件夹里面看看有没有,如果也没有就报错.如果引入的模块名是相对路径,但没有写具体的后缀名的话,那么会先尝试找找有没有.js后缀的,如果没有再看看有没有.json的,还没有就继续尝试别的后缀,如果都没有就报错

  • 简述一下什么是事件循环 eventloop

    js是单线程 所以所有的任务都要排队执行，任务分为两种：宏任务、微任务

    1. 执行宏任务队列中第一个任务，执行完后移除它
    2. 执行所有的微任务，执行完后移除它们
    3. 执行下一轮宏任务（重复步骤2）

  • 什么是事件驱动/和非阻塞I/O模型?

    • 事件驱动就是当事件发生时候才会调用的回调函数，这种函数的执行方式叫做事件驱动。
    • 非阻塞I/o是程序执行过程中，i/o操作不会阻塞程序的执行，也就是在i/o操作的同时，继续执行其他代码。

  • 简单描述一下为什么会产生跨域问题,该如何解决?

    不同源的ajax请求

    jsonp或者cors

  • jsonp的原理是什么?

    script 标签 src 属性中的链接却可以访问跨域的js脚本，利用这个特性，服务端不再返回JSON格式的数据，而是返回一段调用某个函数的js代码，在src中进行了调用，这样实现了跨域

  • express是什么?还使用过类似的东西吗?

    Express 是一个基于 Node.js 平台的极简、灵活的 web 应用开发框架，它提供一系列强大的特性，帮助你创建各种 Web 和移动设备应用。

    Koa,Meteor,socketstream

  • 使用node.js还能干些什么?

    开发动态网站和WEB应用
